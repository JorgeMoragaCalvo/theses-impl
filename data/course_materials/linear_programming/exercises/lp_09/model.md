# Solución P9

## Variables de decisión
- $x =$ cantidad de vans grandes.
- $y =$ cantidad de vans pequeñas.

## Función objetivo
- Minimizar $z = 40x + 20y$

## Restricciones
- De capacidad de paquetes $200x + 80y \geq 1200 $
- De presupuesto $40x + 20y \leq 300 $
- De relación de vans $x \leq y$
- De no negatividad $x, y \geq 0 $
