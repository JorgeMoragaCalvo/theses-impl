# Solución P10

## Variables de decisión
- $x_1 =$ número de cajas de tornillos para madera.
- $x_2 =$ número de cajas de tornillos para metal.

## Función objetivo
- Maximizar: $z = 10x_1 + 17x_2$

## Restricciones
- Máquina de ranurar: $3x_1 + 2x_2 \leq 3600$
- Máquina de roscar: $2x_1 + 8x_2 \leq 3600$
- De no negatividad $x_1, x_2 \geq 0 $

## Solución
Los valores que maximizan la función son: $x_1 = 1080, x_2 = 180$
