# Solución P5

## Variables de decisión
- $X =$ kg. fabricado con alambre de aluminio.
- $Y =$ kg. fabricado con alambre de cobre.

## Función objetivo
- Maximizar $z = 0.25X + 0.4Y$

## Restricciones
- $Y \leq 60$
- $5X + 2Y \leq 500 $
- $0.25X + 0.5Y \leq 40 $
- $X, Y \geq 0$
