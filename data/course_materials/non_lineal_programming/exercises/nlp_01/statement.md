# P2 — Optimización con restricciones
Para el problema:

$ min f(x, y) = (x - 2)^2 + (y - 1)^2 $  
$ s.a. \space -y + x^2 \leq 0 $  
$ s.a. \space x + y - 2 \leq 0 $

Se piensa que $ (1, 1) $ es solución óptima.

## Pista
> Recordatorio: condiciones KKT — $\lambda_i g_i(x, y) = 0 $, $\lambda_i \leq 0 $, y $\nabla f = \lambda_1 \nabla g_1 + \lambda_2 \nabla g_2$
