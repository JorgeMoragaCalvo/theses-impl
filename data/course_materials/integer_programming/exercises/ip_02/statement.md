# P2 — Turnos de carteros

Una oficina de correos necesita una cantidad diferente de carteros a tiempo completo para los distintos días de la semana. La cantidad de carteros a tiempo completo necesarios por día, se calcula según el número de artículos a entregar y es el siguiente:
lunes, $17$ carteros; martes, $13$; miércoles $15$; jueves, $19$; viernes, $14$; sábado $16$; domingo $11$.

Según las normas sindicales, cada empleado a tiempo completo debe trabajar $5$ días seguidos y luego descansar $2$ días. Por ejemplo, un empleado que trabajar de lunes a viernes debe descansar sábado y domingo. El servicio postal desea cubrir sus necesidades diarias y emplear únicamente a personal a tiempo completo

## Lo que se pide
1. Formular un modelo matemático que el servicio postal pueda utilizar para obtener el número mínimo de carteros a tiempo completo con los que debe trabajar.
2. Calcular las dos primeras tablas del método Simplex que se requieren para obtener la solución del problema.
