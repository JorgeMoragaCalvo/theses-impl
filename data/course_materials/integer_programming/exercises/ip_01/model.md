# P1 — Modelo

## Solución
1. Igualar las desigualdades
$$x_1 + 4x_2 = 12 \space (1)$$
$$4x_1 + x_2 = 5 \space (2)$$
2. Resolver el sistema
    - Multiplicando $(2)$ por $-4$
$$x_1 + 4x_2 = 12 \space (1)$$
$$-16x_1 + -4x_2 = -20 \space (2)$$
    - Sumando $(1)$ y $(2)$
$$-15x_1 = -8$$
    - $\therefore x_1 = \frac{8}{15} \approx 0.5333$
    - Reemplazando $x_1$ en $(1)$
$$\frac{8}{15} + 4x_2 = 12$$
$$4x_2 = 12 - \frac{8}{15}$$
$$x_2 = \frac{172}{60} = \frac{43}{15} \approx 2.86666$$
    - $\therefore$ la intersección es: $(\frac{8}{15}, \frac{43}{15})$.
