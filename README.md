# Pantalla principal
A la izquierda se encuentra `login` o `register` que cuenta con los campos:
  - Nombre
  - Email
  - Password
  - Repetir Password

Como ya estoy registrado. Paso al `login`.

# Pantalla de chat
`Chat` del tutor donde se pueden realizar preguntas. Se debe elegir el tema. La barra lateral izquierda permite elegir entre los siguientes agentes:
  - Operations Research
  - Mathematical Modeling
  - Linear Programming
  - Integer Programming
  - Nonlinear Programming

Actualmente, se encuentran habilitados `Mathematical Modeling` y `Linear Programming`. Falta implementar los otros agentes.
